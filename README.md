# 🚦 Urban Mobility vs. Economic Indicators in Latin America

## 📌 Project Overview

This project analyzes the relationship between urban traffic congestion and economic indicators across major Latin American cities.

The objective was to clean, transform, merge, and analyze data from multiple sources using Python to determine whether cities with higher GDP per capita tend to experience greater traffic congestion.

---

## 🎯 Objectives

- Clean and prepare traffic and economic datasets.
- Standardize column names and data types.
- Merge datasets using city and year.
- Perform exploratory data analysis (EDA).
- Visualize the relationship between congestion and GDP per capita.
- Export a clean dataset for future analysis.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📂 Datasets

- TomTom Urban Traffic Dataset
- OECD City Economy Dataset

---

## 📈 Analysis Performed

The project included:

- Data cleaning
- Missing value handling
- Column renaming
- Data type conversion
- Dataset merging (INNER JOIN)
- Descriptive statistics
- Data visualization
- Interpretation of results

---

## 📊 Key Findings

The analysis suggests that there is **no clear linear relationship** between GDP per capita and traffic congestion among the analyzed cities. Although some high-income cities experience significant congestion, the overall pattern indicates that additional factors likely influence urban mobility.
